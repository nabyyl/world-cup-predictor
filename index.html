Office World Cup Prediction League

Scoring Rules

5 pts Exact score
4 pts Correct result + goal difference
3 pts Correct result: winner / draw

You can change your prediction unlimited times before kickoff. The last saved prediction before the match starts is the one that counts.

Predictions are auto-locked the moment a match starts — both in the UI and at the database level (Supabase RLS rejects edits after kickoff). Admin can reopen a match using override only if needed.

Live scores and kickoff timing are sourced from the FIFA-aligned OpenFootball feed. Use the "View on Google" link on any match to open Google's live World Cup card for that fixture.
